# WattsNear website

Static marketing site for WattsNear, ready for GitHub Pages.

## What’s included

- `index.html` — landing page
- `privacy.html` — privacy policy
- `terms.html` — terms of service
- `style.css` — responsive styling
- `script.js` — calculator, theme toggle, and waitlist placeholder
- `.nojekyll` — disables Jekyll processing on GitHub Pages

## Deploying to GitHub Pages

1. Push this repo to GitHub.
2. In the repo settings, enable GitHub Pages.
3. Select the `main` branch and the root folder.
4. Save.

After GitHub finishes deploying, the site will be available at the repo’s Pages URL.

## Notes

The waitlist form stores the email locally in the browser only. It is a placeholder until a real backend is connected.
